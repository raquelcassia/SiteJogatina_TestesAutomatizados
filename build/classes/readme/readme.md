# Teste Automatizado - Site Jogatina

##Roteiro:

###Primeira Etapa

- Acessar o site:  https://www.jogatina.com/
- Clicar no botão CADASTRE-SE
- Inserir e-mail 
- Inserir senha 
- Clicar no botão CRIAR CONTA
- Registrar email e senha no Log

###Segunda Etapa

- Clicar para abrir menu vertical
- Clicar em ALTERAR IMAGEM
- Selecionar icone
- Clicar em ALTERAR IMAGEM
- Buscar o texto de confirmação e Testar 
- Registrar Resultado no Log
- Fechar pop-up	

###Terceira Etapa

- Clicar para abrir menu vertical
- Clicar em ALTERAR APELIDO
- Inserir novo apelido
- Clicar em ALTERAR APELIDO
- Buscar o texto de confirmação e Testar 
- Registrar Resultado no Log
- Fechar pop-up

### Quarta Etapa

- Clicar para abrir menu vertical
- Clicar em SAIR

### Quarta Etapa

- Clicar em icone JOGATINA para voltar pra pag inicial
- Clicar em icone BINGO para voltar pra pag do Bingo
- Verificar se a imagem do print está com o link correto
- Registrar resultado no log.
